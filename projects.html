Lawrence Berkeley National Lab

I've been designing and developing UIs for scientists for over 15 years. From the first UI I designed and developed, a Java based desktop application for tracking supernovae, to broader iniatives, such as creating a design system for science web UIs, I've worked closely with cross functional teams to create software that accomplished mission critical science.

STRUDEL Design System

The STRUDEL Design System provides a set of high level templates for designing and implementing user interfaces for scientific software. This case study describes how from a white paper on science design systems emerged an AI empowered community of practice around science UIs.

How AI, conceptual models, and UX can help novice users develop code

More people are trying to use agenic AI in order to work with unfamiliar coding libraries and languages. But it can be difficult for users to understand the code changes these AIs agents are proposing. In the PrOMMiS project, we are designing an AI aware and UX informed domain specific conceptual model to help users verify AI suggestions.
